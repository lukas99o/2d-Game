# 3. Enkel level

## Mål
Skapa en enkel testlevel med plattformar, mark och lite höjdskillnader för att testa rörelse och kollision.

## Innehåll
- Platt mark som grund
- Några plattformar på olika höjder
- Hinder eller gaps för att testa hoppmekaniken
- Enkel visuell stil (kan vara placeholder-grafik)

## Tekniska noter
- Tilemap eller hårdkodade plattformsrektanglar till en början
- Kameran följer spelaren horisontellt
- Tydliga gränser för levelns start och slut
