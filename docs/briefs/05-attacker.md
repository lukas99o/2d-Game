# 5. Attacker

## Mål
Ge spelaren möjlighet att attackera fiender med ett enkelt attacksystem.

## Funktioner
- Enkel attack (slag eller projektil)
- Träff-detektion mot fiender (hitbox)
- Fienden tar skada eller dör vid träff
- Attack-animation för spelaren

## Tekniska noter
- Attackknapp triggar en kortvarig hitbox framför spelaren
- Cooldown mellan attacker
- Projektil (om tillämpligt): rör sig i en riktning och förstörs vid träff eller väggkollision
