# 1. Rörelssystem med kollision

## Mål
Implementera ett grundläggande rörelssystem för spelaren med kollision mot marken och plattformar.

## Funktioner
- Gå åt vänster och höger
- Sprint (håll knapp för att springa snabbare)
- Hoppa (enkel hopp, eventuellt dubbelhoppa senare)
- Grundläggande kollision med marken och plattformar

## Tekniska noter
- Hantera input via tangentbord
- Använd enkel AABB-kollision (axis-aligned bounding box)
- Gravitation och vertikal hastighet för hopp
